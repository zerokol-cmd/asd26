# asd26 - Listen audio from PC without speakers!
Low-latency <30ms audio transport via local WI-FI. Uses UDP (for lossy internet transport) and Opus for audio encoding.

# Directories
* app - an android app for the server in Java. 
* client - Windows Client written in C# uses `NAudio` and `WASAPI` to capture audio. And `Concentus` for opus support.
